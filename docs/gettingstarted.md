## Requirements

You will need the following:
- Node.JS
- React.JS
- TypeScript
- Go
- Docker


## Installation
Head over to our [repo](https://github.com/2-of-clubs/2ofclubs)

Type in the following:

```
git clone https://github.com/2-of-clubs/2ofclubs
npm install
npm run
```
