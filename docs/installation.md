## Requirements

- Node.js

## Installation
### Frontend
```
git clone https://github.com/2-of-clubs/2ofclubs-server
npm install
npm start
```
### Backend
```
git clone https://github.com/2-of-clubs/2ofclubs-server
cd 2ofclubs-server
...
```
